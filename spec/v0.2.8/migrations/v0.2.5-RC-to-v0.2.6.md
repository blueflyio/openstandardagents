## Overview

This guide helps you migrate your OSSA agent manifests from v0.2.5-RC to v0.2.6.

## Compatibility

- **Backward Compatible**: Yes (with v0.2.5-RC, v0.2.4, v0.2.3, v0.2.2)
- **Breaking Changes**: None expected
- **Migration Required**: Update `apiVersion` field

## Migration Steps

### 1. Update apiVersion

Change the `apiVersion` field in your manifest:

```yaml
# Before (v0.2.5-RC)
apiVersion: ossa/v0.2.5-RC

# After (v0.2.6)
apiVersion: ossa/v0.2.6
```

### 2. Update Schema References

If you reference the schema directly:

```json
{
  "$schema": "https://openstandardagents.org/schemas/v0.2.6/agent.json"
}
```

### 3. Validate Your Manifest

After updating, validate your manifest:

```bash
npm install @bluefly/openstandardagents@latest
ossa validate your-agent.yaml
```
