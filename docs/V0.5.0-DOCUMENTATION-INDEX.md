## Code Examples Index

### Export Examples

**Basic Export:**
```bash
ossa export agent.ossa.yaml --platform npm
ossa export agent.ossa.yaml --platform langchain
ossa export agent.ossa.yaml --platform anthropic
```

**Export with API:**
```bash
ossa export agent.ossa.yaml --platform npm --with-api
ossa export agent.ossa.yaml --platform langchain --with-api
ossa export agent.ossa.yaml --platform anthropic --with-api
```

**Export with Skills:**
```bash
ossa export agent.ossa.yaml --platform npm --skill
```

### Usage Examples

**npm Package Usage:**
```javascript
import agent from '@ossa/my-agent';
console.log(agent.metadata.name);
const manifest = agent.manifest();
```

**LangChain Usage:**
```python
from agent import MyAgent
agent = MyAgent()
result = agent.run("test input")
```

**Anthropic Usage:**
```typescript
import { AnthropicAdapter } from './runtime';
const adapter = new AnthropicAdapter(manifest);
const response = await adapter.execute([...]);
```

### API Usage Examples

**Execute via HTTP:**
```bash
curl -X POST http://localhost:3000/api/v1/execute \
  -H "Content-Type: application/json" \
  -d '{"input":"test"}'
```

**Streaming:**
```bash
curl -N -X POST http://localhost:3000/api/v1/chat/stream \
  -H "Content-Type: application/json" \
  -d '{"messages":[{"role":"user","content":"hello"}]}'
```

## OSSA Manifest Examples

### Basic Agent

```yaml
apiVersion: ossa/v0.5.0
kind: Agent
metadata:
  name: my-agent
  version: 1.0.0
spec:
  role: Assistant
  llm:
    provider: anthropic
    model: claude-3-5-sonnet-20241022
    temperature: 0.7
    maxTokens: 2048
```

### Agent with Cost Optimization

```yaml
apiVersion: ossa/v0.5.0
kind: Agent
metadata:
  name: optimized-agent
spec:
  llm:
    provider: anthropic
    model: claude-3-5-sonnet-20241022
    temperature: 0.3
    maxTokens: 1024

  extensions:
    anthropic:
      prompt_caching:
        enabled: true

  token_budget:
    max_total_tokens: 100000
    reset_interval: daily
```

### Agent with Tools

```yaml
apiVersion: ossa/v0.5.0
kind: Agent
metadata:
  name: tool-agent
spec:
  role: Developer Assistant
  tools:
    - name: analyze_code
      description: Analyze code for issues
      parameters:
        - name: code
          type: string
          required: true
        - name: language
          type: string
          required: true
```

## Deployment Examples

### Docker

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --production
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

### Kubernetes

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: my-agent
spec:
  replicas: 3
  template:
    spec:
      containers:
      - name: agent
        image: my-agent:latest
        ports:
        - containerPort: 3000
```

### Serverless (AWS Lambda)

```typescript
export const handler = async (event: any) => {
  const { input } = JSON.parse(event.body);
  const result = await agent.execute(input);
  return {
    statusCode: 200,
    body: JSON.stringify(result)
  };
};
```
