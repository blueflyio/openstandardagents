# Installation Guide - openapi-ai-agents-standard

## Prerequisites

- System requirements as specified in project documentation

## Installation

### Standard Installation

```bash
npm install
```

### Development Installation

```bash
# Clone the repository
git clone <repository-url>
cd openapi-ai-agents-standard

# Install dependencies
npm install
npm run build
```

## Verification

To verify the installation:

```bash
npm test
npm run lint
```
