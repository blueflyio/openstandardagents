# CLI Tests

This directory contains test files for the OSSA CLI implementation.

## Test Structure

- Unit tests for CLI commands
- Integration tests for agent validation
- Schema validation tests
- Template generation tests

## Running Tests

```bash
# Run all tests
npm test

# Run tests in watch mode
npm run test:watch

# Run with coverage
npm run test:coverage
```

Tests use Vitest framework as configured in package.json.
