## Release Process

1. **Merge to main:**
   - Create MR from development → main
   - Review and merge

2. **Execute Release:**
   - Go to GitLab CI/CD → Pipelines
   - On main branch, manually trigger `release:main:minor` job
   - Monitor pipeline execution

3. **Verify Release Artifacts:**
