# OSSA Test Command

Runs OSSA test suite with appropriate coverage.

## Usage
`/ossa-test [test-type]`

## Behavior
- Runs unit, integration, or e2e tests
- Validates schema and examples
- Checks test coverage
- Reports failures with context

## Test Types
- `unit` - Unit tests only
- `integration` - Integration tests
- `e2e` - End-to-end tests
- `all` - Full test suite (default)
- `coverage` - Test with coverage report

## Example
```
/ossa-test integration
```

## Tool Usage
- Terminal: Run test commands
- Read File: Examine test files
- Search: Find related test cases
