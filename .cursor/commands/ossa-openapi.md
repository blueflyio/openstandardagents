# OSSA OpenAPI Command

Validates and generates OpenAPI extensions for OSSA.

## Usage
`/ossa-openapi [action]`

## Behavior
- Validates OpenAPI specs in `openapi/` directory
- Generates TypeScript types from OpenAPI
- Validates OSSA OpenAPI extensions
- Ensures OpenAPI compliance

## Actions
- `validate` - Validate all OpenAPI specs
- `generate` - Generate types from OpenAPI
- `extensions` - Validate OSSA OpenAPI extensions

## Example
```
/ossa-openapi validate
```

## Tool Usage
- Read File: Examine OpenAPI specs
- Terminal: Run validation/generation commands
- Search: Find OpenAPI patterns
