# OSSA Refactor Command

Refactors OSSA codebase with TDD principles.

## Usage
`/ossa-refactor [target]`

## Behavior
- Focuses on code structure and organization
- Improves readability and maintainability
- Does NOT add new features
- Ensures tests pass after refactoring
- Follows SOLID principles

## Example
```
/ossa-refactor src/services/
```

## Tool Usage
- All tools enabled for refactoring
- Must run tests after changes
- Must maintain backward compatibility
