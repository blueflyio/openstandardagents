# Swarm Orchestration Agents

This project integrates with the agent-buildkit swarm orchestration system.

## Available Commands

- `buildkit swarm orchestrate` - Orchestrate enhancements
- `buildkit swarm status <sessionId>` - Check progress
- `buildkit swarm enhance --all` - Quick ecosystem upgrade

## Agent Configuration

See `.claude/swarm-config.json` for project-specific settings.
