# CLAUDE.md - OSSA Project Configuration

This file provides guidance to Claude Code when working with the OSSA specification.

## Project Overview
OSSA (Open Standards for Scalable Agents) is the authoritative specification standard for AI agents.

## Key Focus Areas
- Specification definition and standards
- Production runtime orchestration  
- Global agent registry and discovery
- Compliance and certification
- Federation and multi-tenancy

## Development Guidelines
- Follow TDD methodology
- Maintain specification/implementation separation
- Use semantic versioning
- Document all API changes
